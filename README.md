# iloom-workspace

> 일룸 마케팅팀 AI 전환(AX) 워크스페이스

---

## Quick Start

### 1. Clone
```bash
git clone [repo-url]
cd iloom-workspace
```

### 2. Claude Code 실행
VS Code 또는 터미널에서 해당 폴더 열기

### 3. 첫 대화
```
"이 폴더 구조 분석해줘"
```

---

## 폴더 구조

```
iloom-workspace/
├── CLAUDE.md          # 워크스페이스 컨텍스트 (필독)
├── 00-inbox/          # 임시 저장
├── 00-system/         # 템플릿, 가이드
├── 10-projects/       # 진행 중인 프로젝트
├── 20-operations/     # 운영 업무
├── 30-knowledge/      # 지식 아카이브
├── 40-personal/       # 개인 기록 (daily, weekly)
├── 50-resources/      # 참고 자료
└── 90-archive/        # 완료/보관
```

---

## 핵심 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/daily-note` | 오늘의 Daily Note 생성 |
| `/todo` | 할 일 추가 |
| `/idea` | 아이디어 저장 |
| `/thinking-partner` | 브레인스토밍 |

---

## 팀별 미션

| 팀 | 미션 | 세션 |
|----|------|------|
| 마케팅 | 버벌소스 생성, 채널 관리 | Session 4 |
| 고문정 | 소비자 조사, 리뷰 분석 | Session 3 |
| 브경디 | 이미지소스 생성 | Session 4 |

---

## 사용 팁

1. **Escape**: 작업 중단
2. **/clear**: 컨텍스트 초기화
3. **@ 참조**: 파일 직접 참조 (`@파일명.md`)
4. **Tab**: 자동완성
