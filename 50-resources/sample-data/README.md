> "이번 주 금요일에 2분기 마케팅 성과 리뷰 회의야.
> 본부장한테 보고할 자료 준비해야 하는데, 데이터 분석 좀 도와줄 수 있어?
>
> 영업팀, 마케팅팀, 고문정팀에서 데이터 받아놨으니까
> 이거 분석해서 인사이트 뽑아줘.
>
> 본부장은 숫자보다 '그래서 뭘 해야 하는데?'를 좋아해.
> 액션 아이템까지 정리해주면 좋겠어."

## 미션

### 미션 1: 제품 판매 실적 분석

**팀장의 요청:**
> "2분기 제품별 판매 실적을 분석해서 슬라이드 3장 분량으로 요약해줘.
> 본부장이 좋아하는 건 숫자보다 인사이트야."

**분석 파일**: `sample_sales_data.csv`

**분석 포인트**:
- 월별 매출 추이 (4월 → 5월 → 6월)
- 카테고리별 성과 비교 (침대/책상/아이방)
- 채널별 실적 (온라인/오프라인)
- 제품별 매출 비중

**질문 예시**:
```
"sample_sales_data.csv를 분석해서 2분기 판매 현황을 알려줘"
"월별 매출 추이가 어때?"
"카테고리별 매출을 비교해줘"
"온라인과 오프라인 중 어디가 더 매출이 높아?"
```

---

### 미션 2: 고객 현황 점검

**팀장의 요청:**
> "VIP 고객 중 비활성화된 고객이 있어. 본부장한테 이 얘기 나올 거야.
> 왜 구매가 줄었는지, 다시 활성화할 수 있는지 분석해봐."

**분석 파일**: `customer_data.csv`

**분석 포인트**:
- VIP 고객 식별 (누적매출 상위 20%)
- 비활성 고객 분석 (마지막 구매일 기준)
- 고객 유형별 분포
- 고객 이탈 패턴

**질문 예시**:
```
"customer_data.csv에서 VIP 고객 (상위 20%)을 찾아줘"
"비활성 고객은 몇 개월 전에 마지막 구매를 했어?"
"고객 유형별로 분류해줘"
"이탈 위험이 있는 고객은 누구야?"
```

---

### 미션 3: 마케팅 채널 ROI 검토

**팀장의 요청:**
> "마케팅 예산을 많이 썼는데 효과가 있었나 봐야 해.
> 채널별로 ROI 비교해서 다음 분기 예산 조정 근거를 만들어줘."

**분석 파일**: `marketing_campaign_data.csv`

**분석 포인트**:
- 채널별 ROI 비교
- 비용 대비 전환율 분석
- 효율적인 채널 vs 비효율적인 채널
- 다음 분기 예산 배분 제안

**질문 예시**:
```
"marketing_campaign_data.csv에서 ROI가 가장 높은 채널은?"
"예산 대비 효율이 낮은 채널은 어디야?"
"다음 분기에 예산을 늘려야 할 채널과 줄여야 할 채널을 추천해줘"
"월별로 마케팅 성과가 어떻게 변했어?"
```

---

### 미션 4: 재고 관리 이슈

**팀장의 요청:**
> "물류팀에서 재고 이슈가 있다고 해.
> 인기 상품 재고부족 항목 정리하고, 발주 우선순위 정해줘."

**분석 파일**: `inventory_data.csv`

**분석 포인트**:
- 재고 상태별 품목 분류 (긴급/재고부족/품절)
- 긴급 발주 필요 품목
- 30일 판매량 대비 재고일수
- 제품별 재고 회전율

**질문 예시**:
```
"inventory_data.csv에서 재주문이 필요한 품목을 찾아줘"
"품절된 제품은 뭐야?"
"긴급 발주가 필요한 품목의 우선순위를 정해줘"
"재고가 넉넉한 품목은 뭐야?"
```

---

### 미션 5: 고객 만족도 분석

**팀장의 요청:**
> "고문정팀에서 고객 만족도 설문 결과 공유받았어.
> 본부장이 고객 반응에 관심 많아서 이것도 넣어야 해."

**분석 파일**: `customer_survey_data.csv`

**분석 포인트**:
- 제품 카테고리별 만족도 비교
- 개선이 필요한 영역 (품질, 배송, 서비스 등)
- 채널별 고객 경험 차이
- 추천의향(NPS) 분석

**질문 예시**:
```
"customer_survey_data.csv를 분석해서 카테고리별 만족도를 비교해줘"
"가장 개선이 필요한 영역은 뭐야?"
"온라인/오프라인 고객 만족도 차이가 있어?"
"추천의향이 낮은 제품은 어떤 거야?"
```
